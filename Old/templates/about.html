PolySymph

Music composition and experimentation reimagined.

PolySymph is a revolutionary music composition and experimentation app, designed to replicate the quintessential one-man-band experience necessary for every aspiring modern musician.


PolySymph can implement the following features (Upcoming features are marked in *red):


  • Compositional Tool - reimagination of the way we structure our music compositional process
  • Music Theorist - synthesizes simple major/minor chords automatically to keep you in the flow
  • Piano - interactive realistic interface display that can output sounds in real time
  • *Synthesizer - generate an original library of sounds that can map to the keyboard
  • *Other Instruments - clean visual graphics will display other instrument families

All sounds synthesized through digital signal processing with the Python packages pydub and thinkdsp.
